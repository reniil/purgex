# Push Instructions

Create the repo on GitHub first:
1. Go to https://github.com/new
2. Name: pulsechain-explorer
3. Make it public
4. Click Create Repository
5. Do NOT initialize with README (we have one)

Then run these commands:
```bash
cd /home/ralph/.openclaw/workspace/pulsechain-explorer
git remote set-url origin https://github.com/reniil/pulsechain-explorer.git
git push -u origin master
```

If prompted for credentials, use your GitHub token.
